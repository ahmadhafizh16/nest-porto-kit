## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository. Implement [Porto Design Patern](https://github.com/Mahmoudz/Porto) with additional Service Layer before action, because I haven't decided where to validate the Request payload, so currently I use the service layer to validate the Request.

## Project setup

```bash
$ npm install
$ npm run db:migrate
$ npm run db:migrate
```

## Generating Basic File for New Module
```bash
$ chmod +x ./create-module.sh 
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
